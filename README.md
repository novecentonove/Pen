# tauri

A md/html editor made with Vite, Vue 3 and TipTap

https://tiptap.dev/ <br>
https://vuejs.org/ <br>
https://vitejs.dev/ <br>


## Project Setup

```sh
npm install
```

### check dependencies
```https://tauri.app/v1/guides/getting-started/prerequisites/```

### Update Cargo
```sh
cd src-tauri
cargo build
 ```

### Compile and Hot-Reload for Development

```sh
npm run tauri dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run tauri build
```
